EN 301-549 Non-Web Software (Chapter 11)

Learning objective:
Accessibility requirements suitable for public procurement of ICT products and services in Europe" specifies the Functional Accessibility Requirements applicable to ICT products and services. It contains a description of the test procedures and evaluation methodology for each accessibility requirement. The requirements are suitable for use in public procurement within Europe.
