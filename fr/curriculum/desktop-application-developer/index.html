Accessibilité numérique : Favoriser la participation à la société de l'information (lien externe) (en anglais seulement)

Objectif d’apprentissage :
Ce cours vous permettra de comprendre les obstacles que doivent surmonter les personnes ayant une incapacité sensorielle, physique ou cognitive lorsqu'elles utilisent les technologies numériques. Le cours expliquera de quelle manière la conception accessible et inclusive contribue à éliminer bon nombre de ces difficultés.
Niveau :
Débutant
Fournisseur :

Préalables :
Aucun
Durée :
15 heures
Mode de prestation :
Cours en ligne
Estimation du coût :
Gratuit
Note :
actuellement non disponible

Conception de matériel pédagogique accessible [variantes : général, Adobe Captivate, Articulate Storyline, Blackboard] (en anglais seulement)

Objectif d’apprentissage :
Peu importe l'outil que vous utilisez (Adobe Captivate, Articulate Storyline, Studio, Canvas, Lectora, ATutor ou le langage HTML5), nous vous expliquerons en détail la marche à suivre pour créer un contenu qui saura plaire à l'ensemble de votre public cible, tout en vous conformant à la réglementation relative à l'accessibilité pour les personnes ayant un handicap. Le cours portera également sur la norme SCORM sur l'accessibilité et les systèmes de gestion de l'apprentissage (comme Blackboard et D2L). Vous verrez à quel point il est possible de réduire le total de coûts et les efforts déployés tout en produisant de meilleurs résultats d'apprentissage pour tous.
Niveau :
Débutant, Concepteurs de matériel pédagogique
Fournisseur :
David Berman Communications (en anglais seulement)
Préalables :
Aucun
Durée :
Cours d'une journée, d'une demi-journée ou sous forme d'exposé principal. Nous offrons ce cours en l'adaptant sur place aux besoins de votre organisation.
Mode de prestation :
En ligne ou sur place
Estimation du coût :
CAD330$ (rabais de groupe disponible)

Inscrivez-vous pour Conception de matériel pédagogique accessible

T716 : Accessibilité des sites Web de Canada.ca

Remarque : Les participants doivent avoir un compte pour avoir accès au cours.

Objectif d’apprentissage :
Ce cours présente les normes et lignes directrices à ce sujet. Il met l’accent sur les exigences, les rôles et les responsabilités en matière de conformité à ces normes. Les participants apprendront à évaluer les Règles pour l’accessibilité des contenus Web (WCAG) 2.1 et à les appliquer de façon uniforme dans les pages Web de Canada.ca.
Fournisseur :
École de la fonction publique du Canada
Préalables :
Spécialiste en information technique Web
Durée :
15 heures
Mode de prestation :
Cours en salle de classe
Estimation du coût :
Gratuit
Langue :
Anglais et Français

Règles pour l'accessibilité des contenus Web (WCAG) 2.1 (en anglais seulement) Activité adaptée au rythme de l'apprenant – lecture

Objectif d’apprentissage :
Les Règles pour l'accessibilité des contenus Web (WCAG) 2.1 présentent un large éventail de recommandations qui visent à rendre les contenus Web plus accessibles. Ces lignes directrices traitent de l'accessibilité du contenu Web qui est affiché sur les ordinateurs de bureau, les ordinateurs portatifs, les tablettes et les appareils mobiles.
Niveau :
Intermédiaire
Fournisseur :
Consortium World Wide Web (W3C)
Préalables :
Non indiqué
Durée :
Cours adapté au rythme de l'apprenant.
Mode de prestation :
Activité adaptée au rythme de l'apprenant – lecture.
Estimation du coût :
Gratuit
